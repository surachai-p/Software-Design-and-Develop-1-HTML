นางสาวณัฏฐวรรณ ช่างเก็บ

สถาบันเทคโนโลยีพระจอมเกล้าเจ้าคุณทหารลาดกระบัง (สจล.)

ปัจจุบันกำลังศึกษาอยู่ชั้นปีที่ 1 ภาควิชาครุศาสตร์อุตสาหกรรม สาขาเทคโนโลยีคอมพิวเตอร์ ซึ่งเป็นการเรียนที่ผสมผสานระหว่าง ทักษะด้านไอที และ จิตวิทยาการสอน เข้าด้วยกัน

เป้าหมายในตอนนี้คือการเก็บเกี่ยวความรู้ด้านพื้นฐานโปรแกรมมิ่ง
และทำความรู้จักกับเพื่อนใหม่ในรั้วแคแสดให้มากที่สุด

        [ งานอดิเรกที่น่าสนใจสำหรับสายคอมฯ ]
        1. ศึกษาเทรนด์ AI และเครื่องมือดิจิทัลใหม่ๆ 
        2. ออกแบบสื่อการสอนแบบ Interactive (Canva/Figma)
        3. ฝึกเขียน Code พื้นฐานและแก้โจทย์ Logic
        4. ถ่ายภาพและตัดต่อวิดีโอสั้นลง Social Media
