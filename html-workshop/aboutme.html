ประวัติการศึกษา

ปัจจุบันเป็นนักศึกษาชั้นปีที่ 1 สาขาเทคโนโลยีคอมพิวเตอร์

สถาบันการศึกษา: สถาบันเทคโนโลยีพระจอมเกล้าเจ้าคุณทหารลาดกระบัง (KMITL)

งานอดิเรก

ความสนใจของฉันมีดังนี้:

  • การเขียนโปรแกรม (Coding)
  • หุ่นยนต์ (Robotics)
  • เล่นเกมส์

เป้าหมายในอนาคต

เป้าหมายของฉันคือ:

  1. ศึกษาและพัฒนาทักษะในด้าน Coding และ Robotics
  2. สร้าง Code และหุ่นยนต์ที่สามารถช่วยคนได้
  3. กลายเป็นวิศวกรและนักพัฒนาซอฟต์แวร์ที่มีความสามารถ
