Semantic Elements คืออะไร?

Semantic Elements คือ Elements ใน HTML5 ที่มีความหมายในตัวเอง (Meaningful tags) ซึ่งช่วยให้ทั้ง Web Browser และ Developer เข้าใจโครงสร้างของหน้าเว็บได้ชัดเจนยิ่งขึ้น

ตัวอย่างเช่น การใช้ <header> แทนที่จะใช้ <div class="header"> หรือการใช้ <article> แทน <div class="post">

ข้อดีคือช่วยเรื่อง SEO (Search Engine Optimization) ทำให้ Google เข้าใจเนื้อหาเว็บเราได้ดีขึ้น และช่วยเรื่อง Accessibility สำหรับผู้พิการทางสายตาที่ใช้โปรแกรมอ่านหน้าจอ
