ส่วนที่ 2: รายการ

รายการแบบไม่เรียงลำดับ - งานอดิเรก

  • การเขียนโปรแกรม (Coding)
  • หุ่นยนต์ (Robotics)
  • เล่นเกมส์

รายการแบบเรียงลำดับ - ขั้นตอนการศึกษา

  1. ประถมศึกษา (6 ปี)
  2. มัธยมศึกษาตอนต้น (3 ปี)
  3. มัธยมศึกษาตอนปลาย (3 ปี)
  4. อุดมศึกษา (ปจจุบัน)

รายการแบบคำจำกัดความ - ศัพท์เทคโนโลยี

HTML
HyperText Markup Language - ภาษามาร์กอัปสำหรับสร้างหน้าเว็บ
CSS
Cascading Style Sheets - ภาษาสำหรับจัดรูปแบบของหน้าเว็บ
JavaScript
ภาษาโปรแกรมสำหรับการเพิ่มความสามารถแบบอินเตอร์แอ็กทีฟให้กับหน้าเว็บ
API
Application Programming Interface - ส่วนติดต่อสำหรับการเชื่อมต่อโปรแกรม
Database
ฐานข้อมูล - ที่เก็บรวมของข้อมูลที่จัดระบบ
