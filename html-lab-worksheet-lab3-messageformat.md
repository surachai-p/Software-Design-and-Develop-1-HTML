# ใบงานการทดลอง HTML
 
## การทดลองที่ 3: การจัดการข้อความและการจัดรูปแบบ
### ขั้นตอนการทดลอง
1. ทดลองใช้ tag ต่างๆ:
```html
<h1>หัวข้อระดับ 1</h1>
<h2>หัวข้อระดับ 2</h2>
<p>ย่อหน้าปกติ</p>
<p>ข้อความ <strong>ตัวหนา</strong> และ <em>ตัวเอียง</em></p>
<p>ขึ้นบรรทัดใหม่<br>ด้วย br</p>
<hr>
<pre>
    ข้อความที่ต้องการ
    รักษารูปแบบ
    การเว้นวรรค
</pre>
```

### แบบฝึกหัด
1. สร้างหน้าเว็บแนะนำตัวเองที่ประกอบด้วย:
   - ชื่อ-นามสกุล
   - ประวัติการศึกษา
   - งานอดิเรก
   - เป้าหมายในอนาคต
 ข้อกำหนดที่ต้องมี:
   - หัวข้อหลักและหัวข้อย่อย
   - ย่อหน้าที่มีการจัดรูปแบบ
   - การขึ้นบรรทัดใหม่
   - เส้นคั่นระหว่างเนื้อหา
### บันทึกผลการทดลอง
- รหัสเอกสาร HTML ที่เขียน:
```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>ทดลองใช้ tag</title>
</head>
<body>
    <h1>นัทธพงศ์ เหมือนประสาน</h1>
    <h2>มัธยมศึกษาปีที่1-3 โรงเรียนศรีราชา</h2>
    <h2>อาชีวะศึกษาปีที่1-3 วิทยาลัยเทคสัตหีบ</h2>
    <h2>ปัจจุบันศึกษาระดับอุดมศึกษาที่ สถาบันเทคโนโลยีพระจอมเกล้าเจ้าคุณทหารลาดกระบัง</h2>
    <p>งานอดิเรก<br>เวลาว่างก็ชอบ <strong>นอน,ดูหนัง,ฯลฯ ให้บอกทั้งหมดคงจะยากงั้นผมจะบอกสูตรลาบแล้วกัน</strong></p>
    <pre>ส่วนผสม
เนื้อสับ 300 กรัม
น้ำเปล่า ½ ถ้วย
ต้นหอมซอย 2 ต้น
ใบสะระแหน่ 2 ต้น
ผักชีฝรั่งซอย 2 ต้น
หอมแดงซอย 3 กลีบ
พริกป่น 2 ช้อนโต๊ะ
น้ำปลา 2 ช้อนโต๊ะ
น้ำมะนาว 2 ช้อนโต๊ะ
ข้าวคั่ว 2 ช้อนโต๊ะ
น้ำตาล ½ ช้อนชา
พริกแห้งทอดสำหรับตกแต่ง
วิธีทำ
ใส่น้ำเปล่าลงในหม้อ แล้วใส่เนื้อสับลงไปรวนให้สุก
ตักเนื้อที่รวนแล้วใส่ในชามผสม
ใส่เครื่องปรุง พริกป่น น้ำปลา ข้าวคั่ว และน้ำตาล คลุกเค้าให้เข้ากัน
จากนั้นเติมมะนาว ผักชีฝรั่ง หอมแดงซอย ต้มหอม ผักชีฝรั่ง และใบสะระแหน่ จากนั้นคลุกเคล้าให้เข้ากันแล้วใส่ในจานตัวเอง แล้วรับประทานได้ครับ</pre>
<hr><h1>เป้าหมายในอนาคต</h1>
<p>ผมก็อยากจะเป็นคนรวยแหละ</p>
<p>แล้วผมก็จะไปลง สส. พรรคพี่เต้เพราะผมอยากไปเดินเซ็นทรัลสาขาดาวอังคาร</p>
</body>
</html>
```
