# ใบงานการทดลอง HTML
 
## การทดลองที่ 3: การจัดการข้อความและการจัดรูปแบบ
### ขั้นตอนการทดลอง
1. ทดลองใช้ tag ต่างๆ:
```html
<h1>หัวข้อระดับ 1</h1>
<h2>หัวข้อระดับ 2</h2>
<p>ย่อหน้าปกติ</p>
<p>ข้อความ <strong>ตัวหนา</strong> และ <em>ตัวเอียง</em></p>
<p>ขึ้นบรรทัดใหม่<br>ด้วย br</p>
<hr>
<pre>
    ข้อความที่ต้องการ
    รักษารูปแบบ
    การเว้นวรรค
</pre>
```

### แบบฝึกหัด
1. สร้างหน้าเว็บแนะนำตัวเองที่ประกอบด้วย:
   - ชื่อ-นามสกุล
   - ประวัติการศึกษา
   - งานอดิเรก
   - เป้าหมายในอนาคต
 ข้อกำหนดที่ต้องมี:
   - หัวข้อหลักและหัวข้อย่อย
   - ย่อหน้าที่มีการจัดรูปแบบ
   - การขึ้นบรรทัดใหม่
   - เส้นคั่นระหว่างเนื้อหา
### บันทึกผลการทดลอง
- รหัสเอกสาร HTML ที่เขียน:
```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
</head>
<body>
    <h1>ชื่อ-นามสกุล</h1>
    <h2>ชิษณุพัศก์ วายุสุวรรณวิทย์</h2>
    <p>นักศึกษาชั้นปีที่ 1 สาขา <strong>คณะครุศาสตร์อุตสาหกรรมและเทคโนโลยี</strong></p>

    <hr>

    <h1>ประวัติการศึกษา</h1>
    <h2>การศึกษาปัจจุบัน</h2>
    <p>สถาบันเทคโนโลยีพระจอมเกล้าเจ้าคุณทหารลาดกระบัง (KMITL)<br>
    สาขา เทคโนโลยีคอมพิวเตอร์, ชั้นปีที่ 1 <em>(ปี 2568-ปัจจุบัน)</em></p>
    
    <h2>การศึกษามัธยมศึกษา</h2>
    <p>โรงเรียน <strong>โรงเรียนสาธิตมหาวิทยาลัยรามคำแหง</strong><br>
    ตัดสินใจเลือก <em>ศิลปภาษา-อังกฤษ</em>

    <hr>

    <h1>งานอดิเรก</h1>
    <p>ความสนใจหลักของฉัน:</p>
    <ul>
        <li><strong>การเขียนโปรแกรม</strong> - <em>Python</em></li>
        <li><strong>DevOps</strong> - <em>Docker, Kubernetes, CI/CD</em></li>
        <li>อ่านหนังสือเกี่ยวกับเทคโนโลยี</li>
    </ul>

    <hr>

    <h1>เป้าหมายในอนาคต</h1>
    <h2>ระยะสั้น (1-2 ปี)</h2>
    <p>สำเร็จการศึกษาปริญญาตรี  <em>ได้รับประสบการณ์ฝึกงาน</em> ที่บริษัทพัฒนาซอฟต์แวร์ชั้นนำ</p>

    <h2>ระยะยาว (3-5 ปี)</h2>
    <p>อยากเป็น <strong>DevOps Engineer</strong> ที่เก่งมาก ๆ <br>
</body>
</html>
```
