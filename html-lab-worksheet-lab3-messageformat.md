# ใบงานการทดลอง HTML
 
## การทดลองที่ 3: การจัดการข้อความและการจัดรูปแบบ
### ขั้นตอนการทดลอง
1. ทดลองใช้ tag ต่างๆ:
```html
<h1>หัวข้อระดับ 1</h1>
<h2>หัวข้อระดับ 2</h2>
<p>ย่อหน้าปกติ</p>
<p>ข้อความ <strong>ตัวหนา</strong> และ <em>ตัวเอียง</em></p>
<p>ขึ้นบรรทัดใหม่<br>ด้วย br</p>
<hr>
<pre>
    ข้อความที่ต้องการ
    รักษารูปแบบ
    การเว้นวรรค
</pre>
```

### แบบฝึกหัด
1. สร้างหน้าเว็บแนะนำตัวเองที่ประกอบด้วย:
   - ชื่อ-นามสกุล
   - ประวัติการศึกษา
   - งานอดิเรก
   - เป้าหมายในอนาคต
 ข้อกำหนดที่ต้องมี:
   - หัวข้อหลักและหัวข้อย่อย
   - ย่อหน้าที่มีการจัดรูปแบบ
   - การขึ้นบรรทัดใหม่
   - เส้นคั่นระหว่างเนื้อหา
### บันทึกผลการทดลอง
- รหัสเอกสาร HTML ที่เขียน:
```html
[วางโค้ด HTML ที่นี่]
<!DOCTYPE html>
<html>
<head>
    <title>aboutme</title>
</head>
<body>

    <h1>สวัสดีค่ะทุกคน ยินดีที่ได้รู้จักค่ะ</h1>
    <hr>

    <h2>แนะนำตัว</h2>
    <p>
        เราชื่อนางสาวศศิธร  รอบคอบพรมราช<br>
        ชื่อเล่นปันปัน ค่ะ อายุ 19 ปี<br>
    </p>

    <hr>

    <h2>ประวัติการศึกษา</h2>
    <p>
        กำลังศึกษาอยู่ที่ <strong>สถาบันเทคโนโลยีพระจอมเกล้าเจ้าคุณทหารลาดกระบัง (KMITL)</strong><br>
        คณะ: <em>ครุศาสตร์อุตสาหกรรมและเทคโนโลยี</em><br>
        สาขา: เทคโนโลยีคอมพิวเตอร์
    </p>

    <hr>

    <h2>งานอดิเรก</h2>
    <p>
        - ฟังเพลง,ร้องเพลง<br>
        - เล่นเกม
    </p>

    <hr>

    <h2>เป้าหมายในอนาคต</h2>

    <pre>
เรียนให้จบปริญญาตรีเเละสามารถนำความรู้ที่ได้จากการเรียนสาขานี้นำไปต่อยอดในการทำงานในอนาคตได้ค่ะ
    </pre>

</body>
</html>

```
