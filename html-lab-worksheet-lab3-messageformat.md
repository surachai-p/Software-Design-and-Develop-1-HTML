# ใบงานการทดลอง HTML
 
## การทดลองที่ 3: การจัดการข้อความและการจัดรูปแบบ
### ขั้นตอนการทดลอง
1. ทดลองใช้ tag ต่างๆ:
```html
<h1>หัวข้อระดับ 1</h1>
<h2>หัวข้อระดับ 2</h2>
<p>ย่อหน้าปกติ</p>
<p>ข้อความ <strong>ตัวหนา</strong> และ <em>ตัวเอียง</em></p>
<p>ขึ้นบรรทัดใหม่<br>ด้วย br</p>
<hr>
<pre>
    ข้อความที่ต้องการ
    รักษารูปแบบ
    การเว้นวรรค
</pre>
```

### แบบฝึกหัด
1. สร้างหน้าเว็บแนะนำตัวเองที่ประกอบด้วย:
   - ชื่อ-นามสกุล
   - ประวัติการศึกษา
   - งานอดิเรก
   - เป้าหมายในอนาคต
 ข้อกำหนดที่ต้องมี:
   - หัวข้อหลักและหัวข้อย่อย
   - ย่อหน้าที่มีการจัดรูปแบบ
   - การขึ้นบรรทัดใหม่
   - เส้นคั่นระหว่างเนื้อหา
### บันทึกผลการทดลอง
- รหัสเอกสาร HTML ที่เขียน:
```html
<!DOCTYPE html>
<html lang="th">
<head>
    <meta charset="UTF-8">
    <title>แนะนำตัว - ประภาภรณ์ ภูผาลี</title>
</head>
<body>

    <h1>แฟ้มสะสมผลงาน (Portfolio)</h1>
    <hr>

    <h2>ข้อมูลส่วนตัว</h2>
    <p>
        <strong>ชื่อ-นามสกุล:</strong> นางสาวประภาภรณ์ ภูผาลี <br>
        <strong>รหัสนักศึกษา:</strong> 68030085 <br>
        <strong>สาขาวิชา:</strong> เทคโนโลยีคอมพิวเตอร์
    </p>

    <hr>

    <h2>ประวัติการศึกษา</h2>
    <p>
        ปัจจุบันกำลังศึกษาระดับ <em>ปริญญาตรี</em> สถาบันเทคโนโลยีพระจอมเกล้าเจ้าคุณทหารลาดกระบัง <br>
        มีความสนใจพิเศษในด้าน <strong>การพัฒนาเว็บไซต์ (Web Development)</strong> และการออกแบบ User Interface
    </p>

    <hr>

    <h2>งานอดิเรกและความสนใจ</h2>
    <p>
        ในเวลาว่าง ฉันชอบหาความรู้ใหม่ๆ เกี่ยวกับเทคโนโลยีทางอินเทอร์เน็ต <br>
        และชอบการ <em>ถ่ายภาพ</em> รวมถึงการฝึกเขียน Code เบื้องต้นเพื่อพัฒนาทักษะตนเอง
    </p>

    <hr>

    <h2>เป้าหมายในอนาคต</h2>
    <pre>
        1. พัฒนาทักษะ HTML, CSS และ JavaScript ให้เชี่ยวชาญ
        2. ฝึกฝนการทำโปรเจกต์เว็บไซต์ที่ใช้งานได้จริง
        3. เป็นนักพัฒนาเว็บ (Web Developer) มืออาชีพ
        4. สำเร็จการศึกษาด้วยเกรดเฉลี่ยที่น่าประทับใจ
    </pre>

    <hr>
    <p><em>ขอบคุณที่แวะมาเยี่ยมชมหน้าเว็บของฉันนะคะ!</em></p>

</body>
</html>
```
