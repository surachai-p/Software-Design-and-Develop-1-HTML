# ใบงานการทดลอง HTML

## การทดลองที่ 3: การจัดการข้อความและการจัดรูปแบบ

### ขั้นตอนการทดลอง

1. ทดลองใช้ tag ต่างๆ:

```html
<h1>หัวข้อระดับ 1</h1>
<h2>หัวข้อระดับ 2</h2>
<p>ย่อหน้าปกติ</p>
<p>ข้อความ <strong>ตัวหนา</strong> และ <em>ตัวเอียง</em></p>
<p>ขึ้นบรรทัดใหม่<br />ด้วย br</p>
<hr />
<pre>
    ข้อความที่ต้องการ
    รักษารูปแบบ
    การเว้นวรรค
</pre>
```

### แบบฝึกหัด

1. สร้างหน้าเว็บแนะนำตัวเองที่ประกอบด้วย:
   - ชื่อ-นามสกุล
   - ประวัติการศึกษา
   - งานอดิเรก
   - เป้าหมายในอนาคต
     ข้อกำหนดที่ต้องมี:
   - หัวข้อหลักและหัวข้อย่อย
   - ย่อหน้าที่มีการจัดรูปแบบ
   - การขึ้นบรรทัดใหม่
   - เส้นคั่นระหว่างเนื้อหา

### บันทึกผลการทดลอง

- รหัสเอกสาร HTML ที่เขียน:

```html
<!DOCTYPE html>
<html lang="th">
  <head>
    <meta charset="UTF-8" />
    <title>แนะนำตัว - ศุภโชค หอมสมบัติ</title>
  </head>
  <body>
    <h1>ยินดีต้อนรับสู่หน้าเว็บของศุภโชค</h1>
    <p>
      ยินดีที่ได้รู้จักครับ! ผมชื่อ
      <strong>นายศุภโชค หอมสมบัติ</strong> หรือจะเรียกผมว่า
      <strong>"Pokoman"</strong> ก็ได้ครับ
    </p>

    <hr />
    <h2>ประวัติการศึกษา</h2>
    <p>
      ปัจจุบันกำลังศึกษาอยู่ที่
      <strong>สถาบันเทคโนโลยีพระจอมเกล้าเจ้าคุณทหารลาดกระบัง (KMITL)</strong
      ><br />
      คณะครุศาสตร์อุตสาหกรรมและเทคโนโลยี สาขาวิชาเทคโนโลยีคอมพิวเตอร์
    </p>

    <hr />

    <h2>งานอดิเรกและความสนใจ</h2>
    <p>
      ผมเป็นคนที่ชอบเทคโนโลยีและการสร้างสรรค์ครับ งานอดิเรกหลักๆ ของผมคือ:<br />
      1. <em>การถ่ายภาพ</em> (ผมมีเพจชื่อว่า "hiw.makmakkub" ด้วยนะ)<br />
      2. <em>การออกแบบวงจร Digital Logic</em> และการเล่นกับบอร์ด
      ESP32/ESP8266<br />
      3. <em>การเขียนโปรแกรม</em> เช่น React, TypeScript และการทำระบบ Smart Farm
    </p>

    <hr />

    <h2>เป้าหมายในอนาคต</h2>
    <pre>
        พัฒนาทักษะด้าน Full-stack Developer 
        และสร้างระบบ IoT ที่ช่วยยกระดับ
        การทำเกษตรอัจฉริยะ (Smart Farming) ในประเทศไทย
    </pre>

    <hr />

    <p>ขอบคุณที่เข้ามาเยี่ยมชมหน้าเว็บของผมครับ</p>
    <p>ติดต่อผมได้ที่: <strong>Lat Krabang, Bangkok</strong></p>
  </body>
</html>
```
