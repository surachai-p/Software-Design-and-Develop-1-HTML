บทความหลัก: ทำความเข้าใจ HTML5

บทนำ

HTML5 เป็นเวอร์ชันล่าสุดของภาษา HTML ซึ่งได้แนะนำองค์ประกอบเชิงความหมาย (Semantic Elements) ที่ช่วยให้เบราว์เซอร์และเครื่องมือ SEO เข้าใจโครงสร้างของหน้าเว็บได้ดีขึ้น

Semantic Elements หลัก

ต่อไปนี้คือ Semantic Elements ที่ใช้บ่อยในการสร้างหน้าเว็บ:

  • <header> - ส่วนหัวของหน้า
  • <nav> - เมนูนำทาง
  • <main> - เนื้อหาหลัก
  • <article> - บทความ
  • <section> - ส่วนของเนื้อหา
  • <aside> - เนื้อหาเสริม
  • <footer> - ส่วนท้าย

ประโยชน์ของ Semantic Elements

การใช้ Semantic Elements ให้ประโยชน์หลายประการ:

  1. ช่วยเพิ่มความเข้าใจสำหรับ Search Engines
  2. ส่งเสริมการเข้าถึงสำหรับผู้ใช้งาน Screen Reader
  3. ทำให้โค้ด HTML อ่านง่ายและดูแลรักษาได้ดีขึ้น
  4. เป็นมาตรฐาน HTML5 ที่นำเสนอโดย W3C

ตัวอย่างการใช้งาน

ตัวอย่างของการจัดวางหน้าเว็บใช้ Semantic Elements ที่สำคัญเช่น:

<body>
    <header>
        <h1>ชื่อเว็บไซต์</h1>
        <nav>...</nav>
    </header>
    
    <main>
        <article>
            <h2>หัวข้อบทความ</h2>
            <section>...</section>
        </article>
        
        <aside>
            <h3>เนื้อหาเสริม</h3>
        </aside>
    </main>
    
    <footer>
        <p>ลิขสิทธิ์ © 2024</p>
    </footer>
</body>
